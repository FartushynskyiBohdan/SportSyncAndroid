Design a clean, modern sign-up page.

Use the existing brand design system (colors, typography, buttons, input styles). The visual style should feel sporty, modern, and minimal with rounded UI elements and subtle shadows.

Layout:
- Centered authentication card
- Large welcoming headline
- Minimal fields to reduce friction

Page structure:

Top navigation bar
- Logo on the left
- “Login” link on the right

Main content area (centered card):

Card container
- Rounded corners
- Subtle shadow
- Width ~420px
- Comfortable padding

Content inside the card:

Headline:
“Join the Athlete Community”

Subtext:
“Find training partners, competitors, and meaningful connections.”

Form fields:
1. Email address input
2. Password input

Buttons:
Primary button:
“Create Account”

Secondary link below:
“Already have an account? Log in”

Do not include social login placeholders (Google / Apple / others)

Design details:
- Clear input states (focus, hover)
- Password visibility toggle
- Error message placeholders
- Accessibility-friendly spacing
- Mobile responsive layout

Background:
Use brand colors in a subtle gradient or soft abstract athletic-themed background.

Goal:
A modern, trustworthy sign-up experience that feels similar in polish to high-quality consumer apps.
